This snake is the first true game made with Vectarine, and it is available to tweak and modify in the Vectarine Gallery.

It is a simple game which acts as a learning example for how to make games with Vectarine.

It uses shaders, reads keyboard input, draws text and does all the basic things you'd expect in such a game.
