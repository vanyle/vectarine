---
title: The Gallery
description: What is the Gallery?
---

The gallery is a collection of small projects and games made with Vectarine.

Some games are official vectarine examples, others are just games made for fun.

You can open a pull request to add your own project to the gallery, don't forget to credit yourself and others and link to the source code!

You can also add a description of your project, the history, what it showcases, etc. It's up to you!
